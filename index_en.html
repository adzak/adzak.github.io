Some projects

ChattenBOT

ChattenBOT is a bot that is used on a Discord server. The core functionality of ChattenBOT is to send entertaining images and text. These are generated by different commands given by the members of the server. ChattenBOT is written in Java and uses the JDA API.

SIXTEEN

SIXTEEN is a game I made with two other friends. I was responsible for the programming while my friends worked on the artwork and the story. SIXTEEN is developed in GameMaker Studio and coded in GML.

BlueBear

BlueBear was a website which could generate a slideshow of pictures from Instagram. The user could input a tag and the website fetched images from Instagram with that tag. The user had the ability to choose which pictures which were to be included in the slideshow. BlueBear was meant to be used at events to give event goers an interactive experience. I had several tasks in this project, but the main component I worked on was a filtering system. This filtering system could prevent images with inappropriate character, such as offensive symbols and nudity. The filtering system was coded in PHP.

Defender

Defender is a game where the player is supposed to not let trough any attackers. The attackers move in different speed towards the player and the players task is to block them. If the player lets trough three attacker the game is over. The game was programmed to a ChipKIT and was written in C.
